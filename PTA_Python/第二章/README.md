8.转换函数使用
--
可以使用按位乘然后累加的朴素做法。同时，python中int(x,y)可以将字符串x按y进制转换为数字（y默认取值为10）。此外，bin(x)，oct(x)，以及hex（x）可以将一个十进制数转换为二/八/十六进制的字符串，带有进制前缀。
